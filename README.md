## 프로젝트 동기
> 졸업 전 학우들이 실제로 우리가 만든 서비스를 통해 조금이나마 편리함을 느끼며 실제 서비스 운영 시 생각해야 하는 점들을 느끼고 싶었음.</br>
> 이때 곧 다가올 교내 축제 사이트를 통해 학우들에게 필요한 정보를 쉽게 제공함으로 원하는 바를 이룰 수 있다고 생각되어 시작함.<br/>
> 교내 학우들을 대상으로 수요 조사를 진행했고, 조사 대상 중 약 80%에 해당하는 인원이 축제 웹 사이트가 필요하다고 응답함.

## 주요 기능

### ⭐ 부스 댓글 + 좋아요
 - 인기 있는 부스 확인 가능
 - 부스별 후기 댓글 작성 및 좋아요 기능
 - 악성 댓글 신고 기능
 - 비속어 필터링 기능
### ⭐ 축제 프로그램 및 가수 라인업
 - 시간별 축제 프로그램, 가수 라인업 정보 확인
 - 자신이 기대하는 프로그램 또는 가수 좋아요 기능
### ⭐ 실시간 교통정보
 - 실시간 성환역 상/하행선 교통정보 확인
 - 성환역 시간표 확인
### ⭐ 카카오 로그인
 - 간편한 카카오 로그인을 통한 좋아요, 댓글 참여 가능



</br>

## 소감

- 권영태 : 인가/비인가 사용자 모두 사용가능한 API를 위해 필터 체인 개념을 보다 이해할 수 있었다. 그리고 특정 API를 매 요청 시 반복 조회로 인한 부하를 줄여 더 나은 서비스를 제공하기 위해 캐싱을 도입했고 이를 해결할 수 있었던 점이 기억에 남는다.

- 조아빈 : 서버 부하를 줄이기 위해 테이블 인덱스를 생성하거나 캐싱을 적용하고, 동시에 사용자에게 정확한 정보를 제공하기 위한 동시성과 정합성을 고려하여 적용하는 과정을 통해 실무적인 관점에서 프로젝트를 진행할 수 있었던 점이 좋았다.
